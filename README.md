# EditIndicator
编辑框指示器

## gradle使用：

一、Project下的build.gradle文件下添加

allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
}

二、Module下的build.gradle文件下添加

dependencies {
	        compile 'com.github.Lvluffy:TextIndicator:1.0.1'
}

或者

dependencies {
	        implementation 'com.github.Lvluffy:TextIndicator:1.0.1'
}
